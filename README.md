## Features

### 1. Home Screen
- Shop name and branding display
- Shop address card with tap-to-navigate functionality
- Featured cars section (3 cars displayed)
- Quick access to view all cars

### 2. Cars Catalog
- Complete inventory display (all available cars)
- Search functionality (search by car name or brand)
- Car cards showing image, price, brand, year, color, mileage, and fuel type
- Prices displayed in PKR (Lac/Crore format)

### 3. Car Details
- Full-screen car image
- Price badge display
- Complete car specifications:
  - Color
  - Mileage
  - Fuel Type
  - Transmission
  - Engine Capacity
  - Model Year
- Shop purchase information
- Shop address and contact details

### 4. Contact & Navigation
- **Call Button** - Direct phone call to shop
- **Get Directions** - Opens Google Maps with shop location

### 5. User Interface
- Modern dark theme design
- Bottom tab navigation (Home & Cars)
- Smooth transitions and animations
- Responsive layout for all screen sizes

## Build & Release

### Development
```bash
# Install dependencies
npm install

# Start development server
npx expo start
```

### Build APK (Android)
```bash
# Build APK
eas build -p android --profile preview

# Build AAB for Play Store
eas build -p android --profile production
```

### Build iOS
```bash
eas build -p ios --profile production
```
